# Before publishing

This repository is ready for GitHub/HACS publication, but there is one final thing to update after creating the real GitHub repository:

Edit `custom_components/gemini_usage_monitor/manifest.json` and replace:

- `https://github.com/OWNER/gemini-usage-monitor`
- `https://github.com/OWNER/gemini-usage-monitor/issues`

with your real repository URL.
